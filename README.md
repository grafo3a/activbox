# ActivBox
When completed, this app will be a user-friendly open-source service ticketing tool.

## Tech stack
The following technologies are used in this project:

- Java 17
- Spring Boot 3.2
- Spring Data JPA
- H2 Database (in-memory)
- Thymeleaf
- HTML, CSS & Bootstrap
- JavaScript (expected)
- JUnit 5 (expected)
- Maven
